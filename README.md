# EmotionFlow

EmotionFlow is a computer vision term project that analyzes the emotional flow of a video over time using visual features.

## Current Progress
- Video loading
- Frame extraction with frame skipping
- Frame resizing for performance optimization
- HSV-based color analysis
- Brightness / saturation / color ratio visualization

## Tech Stack
- Python
- OpenCV
- NumPy
- Matplotlib

## Next Steps
- Save color analysis results
- Build temporary emotion scores based on color features
- Add face detection and facial emotion analysis
